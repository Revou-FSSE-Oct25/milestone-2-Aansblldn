# REVOFVN

## 1. Deskripsi

Website **REVOFVN** adalah website portofolio sederhana yang menampilkan studio digital kreatif beserta beberapa mini game berbasis web. Website ini dibuat sebagai bagian dari **Milestone 2** dan berfungsi sebagai showcase proyek.

## 2. Tujuan

* Menampilkan identitas studio REVOFVN
* Menyediakan akses ke proyek mini game
* Menjadi media portofolio berbasis web

## 3. Struktur Halaman

* **Homepage**

  * Logo dan judul REVOFVN
  * Deskripsi singkat studio
  * Daftar game yang dapat diakses
* **Halaman Game**

  * Tebak Angka
  * Clicker Game
  * Gunting Batu Kertas

## 4. Fitur Utama

* Tampilan sederhana dan modern
* Navigasi langsung ke halaman game
* Game interaktif menggunakan JavaScript

## 5. Teknologi

* HTML
* CSS
* JavaScript
* GitHub Pages (hosting)

## 6. Cara Menggunakan

1. Buka website REVOFVN
2. Pilih game yang tersedia di halaman utama
3. Mainkan game sesuai instruksi di layar

## 7. Catatan

Website ini dibuat untuk pembelajaran dan pengembangan skill dasar front-end serta logika JavaScript melalui mini game interaktif.
